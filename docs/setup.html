CCALCIE

Setup

Install CALCIE for macOS

  1. Download the signed DMG from the release page.
  2. Drag CALCIE.app into Applications.
  3. Open CALCIE from Applications, not from the build folder.
  4. Grant Microphone, Accessibility, Screen Recording, and Notifications when prompted.
  5. Use Advanced Options to import ChatGPT memory if you want continuity.

First-run memory import

CALCIE can start empty. If you want it to remember context from ChatGPT, copy the prompt inside Advanced Options, paste it into ChatGPT, then paste the fenced response back into CALCIE.

The import stays local by default in calcie_profile.local.json and .calcie/profile_imports/.

Permissions

  • Microphone: push-to-talk voice capture.
  • Accessibility: controlled keyboard/mouse actions you request.
  • Screen Recording: screen vision and screen memory.
  • Notifications: alerts and update messages.
