Try These First

Starter commands for testers

Use these as known-good prompts when you first open CALCIE. After that, start experimenting in your own words.

macOS: hold Right Option to talk Windows: try F8, Right Ctrl, Right Alt, or Ctrl+Shift+Space

Windows beta does not need the macOS Terminal command. Just extract the ZIP and run Launch CALCIE.bat.

For full AI/chat features, testers should add a free Gemini API key in the top-level .env next to Launch CALCIE.bat. The setup page now shows the exact steps.

01

Menu bar companion

Ask follow-up questions, review responses you missed, open advanced tools, and manage runtime state from the menu bar.

02

Voice + vision

Hold Right Option to talk. Use screen vision for alert-first monitoring and local screen memory when explicitly enabled.

03

CALCIE Player

One reusable app-owned YouTube / YouTube Music surface, avoiding browser tab chaos while supporting media controls.

04

Safe automation

Desktop actions stay gated. Orders, payments, and destructive actions stop for confirmation instead of silently executing.

05

Live utility skills

Search, weather, jobs, sports scores, app launching, coding workflows, and browser handoffs route through the Python runtime.

06

Update aware

The app checks a backend manifest and shows available updates before automatic updates are introduced.
