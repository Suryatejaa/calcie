CALCIE 0.1.0 alpha build 1

The first packaged macOS alpha: menu bar shell, local runtime control, CALCIE Player, screen vision, live utility skills, update manifest checks, and first-run memory import.

Download

Download CALCIE 0.1.0 alpha DMG

Download CALCIE 0.1.0 Windows beta ZIP

This first public macOS alpha is not notarized yet. If macOS blocks the app after install, move CALCIE into Applications and run xattr -dr com.apple.quarantine /Applications/CALCIE.app once in Terminal.

The Windows beta download is a portable ZIP containing CalcieTray.exe and the bundled runtime for tester machines.

Highlights

  • Native macOS menu bar shell with Advanced Options.
  • Right Option push-to-talk.
  • CALCIE Player with one reusable YouTube / YouTube Music window.
  • Screen vision and optional screen memory pipeline.
  • Live IPL score parser using CREX series data.
  • Local ChatGPT memory import flow.
  • DMG build script, release metadata, and update manifest support.

Known alpha limits

  • DMG is an unsigned / not-yet-notarized alpha build, so some Macs will require a one-time quarantine removal command after install.
  • Automatic updates are not enabled yet; update checks are notification-only.
  • Screen memory should remain opt-in for testers.
